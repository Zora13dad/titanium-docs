# Appcelerator CLI 7.1.1.GA - 6 September 2019

Appcelerator CLI 7.1.1.GA is a patch release that includes one improvement and several bug fixes.

As of this GA release, the previous CLI patch release is no longer supported. End of support for this version will be up to 2020-03-06 or until the next patch release. Note: major and minor releases continue to be supported according to their nominal lifetime. See Axway Appcelerator Deprecation Policy and Nominal Lifetimes documents for details.

# Component versions

The following components are shipped with CLI 7.1.1:

Alloy

1.14.1

API Builder

3.2.13

Cloud CLI

2.1.2

Titanium CLI

5.2.1

Daemon

1.1.3

# Improvements

  • ALOY-1697 - Add WebView "onlink" callback support

    • added support for setting onlink in views

  • CLI-1362 - Update to latest acs cli

    • Updated acs to version 2.1.2

  • CLI-1368 - Daemon: update to ioslib 2.5.1 to support apple certificate detection

    • Updated ioslib to 2.5.1 to add support for Apple certificate detection

# Fixed issues

  • ALOY-1698 - Compile: Errors thrown during sourcemap step are not visible to a user

  • CLI-1355 - Android "appc run" will always trigger rebuild if "tiapp.xml" is missing encryption policy property

  • CLI-1359 - When commenting out modules in the Tiapp.xml and running a build all modules move to one line

  • CLI-1363 - Daemon: Can't detect iOS 13 simulators

  • CLI-1365 - Daemon is not shut down when downloading a core via appc use

  • CLI-1366 - Studio restarts daemon during the appc use switching
